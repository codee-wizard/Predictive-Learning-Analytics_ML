# Curated Academic Knowledge Base

Add project-owned academic notes here as `.md` or `.txt` files. The AI Study
Coach loads these files at startup, chunks them, embeds them, and retrieves the
most relevant chunks for student questions.

Recommended file groups:

- `math_grade_8_10.md` for algebra, geometry, trigonometry, statistics, and probability.
- `reading_comprehension.md` for main idea, inference, tone, vocabulary, and passage strategy.
- `writing_skills.md` for paragraph structure, essays, grammar, evidence, and revision.
- `study_skills.md` for active recall, spaced repetition, exam planning, and mistake analysis.
- `curriculum_faq.md` for institution-specific syllabus, exam pattern, grading rules, and allowed resources.

Keep each section factual, concise, and source-safe. Prefer short headings and
clear worked examples because the retriever works best with focused chunks.
