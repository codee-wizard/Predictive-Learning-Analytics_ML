# Algebra, Geometry, And Trigonometry

## Variables

A variable is a symbol that represents an unknown or changing value. Algebra
uses variables to describe patterns and solve problems. The same operation must
be applied to both sides of an equation to keep it balanced.

## Linear Equations

A linear equation has variables to the first power and forms a straight line
when graphed. A one-variable linear equation can be solved by isolating the
variable.

Example: 3x + 5 = 20

Subtract 5 from both sides: 3x = 15.

Divide by 3: x = 5.

## Slope

Slope measures steepness and direction of a line. It is the change in y divided
by the change in x:

slope = rise / run = (y2 - y1) / (x2 - x1)

Positive slope rises left to right. Negative slope falls left to right. Zero
slope is horizontal. Undefined slope is vertical.

## Slope-Intercept Form

The equation y = mx + b is slope-intercept form. The value m is the slope and b
is the y-intercept, where the line crosses the y-axis. This form is useful for
graphing quickly.

## Systems Of Equations

A system of equations contains two or more equations that share variables. The
solution is the value or point that satisfies all equations at once. Common
methods are substitution, elimination, and graphing.

## Inequalities

Inequalities compare expressions using >, <, >=, or <=. They are solved like
equations, except when multiplying or dividing both sides by a negative number,
the inequality sign must reverse.

## Quadratic Equations

A quadratic equation has the form ax^2 + bx + c = 0, where a is not zero.
Quadratics can be solved by factorisation, completing the square, graphing, or
the quadratic formula.

Quadratic formula:

x = (-b +/- sqrt(b^2 - 4ac)) / 2a

The expression b^2 - 4ac is the discriminant. If it is positive, there are two
real roots. If it is zero, there is one repeated real root. If it is negative,
there are no real roots.

## Functions

A function maps every input to exactly one output. Function notation f(x) means
the output of function f for input x. A graph represents a function if it passes
the vertical line test.

## Triangles

The interior angles of a triangle sum to 180 degrees. In a right triangle, the
Pythagorean theorem says:

a^2 + b^2 = c^2

where c is the hypotenuse. This theorem is used to find missing side lengths.

## Similar Triangles

Similar triangles have the same angle measures and proportional side lengths.
They are useful for indirect measurement, scale drawings, and trigonometry.

## Circles

The radius is the distance from the centre to the circle. The diameter is twice
the radius. Circumference is 2 pi r. Area is pi r^2. Arc length and sector area
are proportional to the central angle.

## Coordinate Geometry

The distance between two points can be found using the distance formula, which
comes from the Pythagorean theorem. The midpoint formula averages the x-values
and y-values of two endpoints.

## Trigonometric Ratios

In a right triangle, the basic trigonometric ratios are:

sin(theta) = opposite / hypotenuse

cos(theta) = adjacent / hypotenuse

tan(theta) = opposite / adjacent

SOH-CAH-TOA is a memory aid for these ratios. Trigonometry is used to find
unknown sides or angles when a right triangle is involved.

## Common Algebra Mistakes

Students often distribute incorrectly, combine unlike terms, forget to reverse
an inequality sign after multiplying by a negative, or lose a negative sign
during multi-step work. A good check is to substitute the answer back into the
original equation.
